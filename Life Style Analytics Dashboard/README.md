# Lifestyle Analytics Dashboard

A Tableau-based dashboard analyzing key lifestyle factors including **Occupation, Daily walking Steps, Stress Level, Physical Activity Level, Heart Rate**, and **Quality of Sleep**. This project aims to uncover relationships between work patterns and overall health outcomes.

---

## Project Overview

The Lifestyle Analytics Dashboard provides insights into:  
- Sleep behavior across different occupations  
- Stress patterns and their connection to sleep  
- Influence of physical activity on heart rate  
- Overall lifestyle quality indicators  
- Relationships between behavioral and physiological metrics  

The dashboard helps identify trends that can guide wellness initiatives and personal health improvements.
